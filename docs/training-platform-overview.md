# TrainingKit — "AI in the SDLC"
### Kuruma açacağımız AI Agent platformunun hazırlık eğitimi

**Kimin için?** Tüm SDLC rolleri (11 rol) — yalnızca geliştiriciler değil; Portfolio/Project Manager, Governance, Solution Designer, Release/DevOps/Infra, Security, Architect, Tester.

**Neden?** Platformu açmadan önce herkese **ortak AI okuryazarlığı + güvenli kullanım**. Amaç AI ile yazılım inşa etmeyi *ve* AI'ı günlük işte güvenle kullanmayı öğretmek — bir tehdit değil, kapasite olarak.

**Tek cümle:** Bilgi değil davranış hedefler — eğitim sonunda herkes "kendi işimde ne yapacağım"ı yazar.

---

# Özellikler

- **18 modül × 2 seviye (L1/L2), tam çift dilli (EN + TR).**
- **Rol bazlı patikalar:** L1 tüm modüller herkese zorunlu; L2 rolün alanına göre. Admin panelinden düzenlenebilir (must / recommended).
- **Senaryo bazlı sınav (30 soru)** + her modülü kapsayan, gerçek çeldiricili genel sertifika.
- **Zorunlu tamamlama notu (reflection):** "öğrendiğimle işimde ne yapacağım / değeri ne" — yalnızca admin görür.
- **Giriş ekranı:** CIO mesajı → rol seç → başla / devam et.
- **Admin paneli:** içerik düzenleme, rol patikaları, ilerleme raporu, refleksiyonlar.
- **Leaderboard + rol sertifikası**, rozetler; **deterministik puanlama** (runtime LLM yok → ucuz, güvenli, tekrarlanabilir).
- **Otomatik dağıtım (CI):** içerik / şema / fonksiyon / frontend push ile canlıya.

# Geri bildirim & konumlandırma

**AI-mimar incelemesi (içerik gözden geçirildi):**
- ✅ Doğru irtifa: org geneline açılabilir bir **okuryazarlık + benimseme** programı.
- ✅ Ayrışan yön: **SDLC merceği + rol-eşlemesi + governance/risk/security/cost** entegre.
- 🔜 Öneri: builder-olmayanlar için güvenlik çekirdeği (eklendi) ve platform açılınca **uygulamalı capstone**.

**Anthropic (Skilljar) eğitimiyle kıyas → tamamlayıcı, rakip değil:**
- **Bizde güçlü:** genişlik, rol-bazlılık, governance/adoption, program/LMS katmanı, çift dil.
- **Anthropic'te güçlü:** uygulamalı (gerçek Claude), güncel & vendor-spesifik derinlik.
- **Hamle:** bu eğitim = herkese okuryazarlık katmanı; builder rolleri ayrıca Anthropic Academy'ye yönlendirilir.

**Sonuç:** Kurumun AI Agent platformunu güvenle ve hızla benimsemesi için ortak zemin — açılıştan önce hazır.
